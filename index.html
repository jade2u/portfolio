Goal

Make Spotify Wrapped a little more personal with custom graphics according to genre & cover art.

API

Using the Spotify API, I organized results into 3 categories and 3 time periods.
This served as a basis for the design.

Design

For Artist and Genre cards, I designated 2 colors for each genre. I created algorithms to choose complementing/contrasting colors based on the 2 colors.

For Track cards, I used a color picker algorithm instead.
